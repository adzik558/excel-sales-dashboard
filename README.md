# Dashboard sprzedaży (Excel)  
Projekt przedstawia interaktywny dashboard sprzedaży stworzony w Microsoft Excel z wykorzystaniem tabel przestawnych, wykresów oraz fragmentatorów (slicerów).

## Struktura repozytorium

```plaintext
excel-sales-dashboard/
├── README.md
└── sales_dashboard.xlsx
```

## Wykorzystane narzędzia

- Microsoft Excel
- Power Query 

## Kluczowe elementy

- Stworzenie interaktywnego dashboardu sprzedaży w Excelu
- Integracja danych i ich transformacja (Power Query)
- Analiza KPI oraz trendów sprzedaży
- Zastosowanie tabel przestawnych, slicerów i map
- Zapewnienie interaktywności raportu (filtry globalne)

## Cel projektu

Celem projektu było przećwiczenie analizy danych oraz budowy dashboardu w Excelu, skupiając się na przekształceniu surowych danych w czytelne i użyteczne informacje biznesowe.

## Kluczowe wnioski

- Sprzedaż wykazuje ogólny trend wzrostowy w analizowanym okresie
- Najwyższą sprzedaż generują wybrane podkategorie produktów (np. Phones, Chairs)
- Region West osiąga najwyższe wyniki sprzedaży
- Sprzedaż koncentruje się w ograniczonej liczbie grup produktowych
- Widoczne wahania mogą wskazywać na sezonowość sprzedaży
